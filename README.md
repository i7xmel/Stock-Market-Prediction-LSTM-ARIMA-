# Stock Market Analysis

## Overview
Stock market analysis and visualization for three major companies: Ford (F), Berkshire Hathaway (BRK-A), and JPMorgan Chase (JPM). This project collects, analyzes, and visualizes 6 years of historical stock data.

## Features
- Automated data collection from Yahoo Finance API
- Multi-company comparative analysis
- Data quality validation and cleaning
- Interactive visualizations with Plotly and Matplotlib
- 6-year historical data analysis (2018-2024)

## Usage
The notebook sequentially:
1. Installs required dependencies
2. Fetches 6 years of stock data for F, BRK-A, and JPM
3. Performs data exploration and validation
4. Generates comparative visualizations
5. Analyzes stock performance trends

## Data Analysis
- **Companies**: Ford (F), Berkshire Hathaway (BRK-A), JPMorgan Chase (JPM)
- **Time Period**: 6 years of daily data
- **Data Points**: Open, High, Low, Close prices, Volume, Dividends, Stock Splits
- **Data Quality**: No null values detected in any dataset

## Key Findings
- Significant price scale variation (BRK-A ~$300,000+ vs F and JPM)
- Different dividend patterns across companies
- Clean, complete datasets ready for further analysis
- Clear sector-specific performance trends

## Technologies
- Python, yfinance, Pandas, NumPy
- Matplotlib, Plotly for visualizations
- Jupyter Notebook for interactive analysis
